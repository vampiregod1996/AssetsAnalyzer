功能介绍

AssetsAnalyze 通过分析ipa包中的各种资源大小，协助项目组制定资源优化方案，并最终达到降低包大小的目的。

使用方法

  1. 将ipa包拖动到页面顶部绿色部分，等待上传成功，就可以查看各个tab页下的资源信息
  2. 全部资源:所有资源按从大到小的方式排序显示
  3. Resources资源:所有Resources目录下的资源按大小排序显示
  4. 场景资源:所有场景用到的资源按大小排序显示
